# Environment Manager - จัดการ environment variables

## คำสั่ง
```
/env [action: scan|document|validate|generate]
```

## หน้าที่
จัดการ environment variables ให้ปลอดภัยและมี documentation ครบถ้วน

## Actions

### 1. Scan - ค้นหา env vars ที่ใช้ใน project
```
/env scan
```

### 2. Document - สร้าง documentation
```
/env document
```

### 3. Validate - ตรวจสอบ .env กับ .env.example
```
/env validate
```

### 4. Generate - สร้าง .env.example
```
/env generate
```

## Output Format

### Scan Output
```markdown
# Environment Variables Scan

## Variables Found

### Database
| Variable | Files Used | Required |
|----------|------------|----------|
| DB_HOST | config.php, db.php | ✅ |
| DB_NAME | config.php | ✅ |
| DB_USER | config.php | ✅ |
| DB_PASS | config.php | ✅ |

### Application
| Variable | Files Used | Required |
|----------|------------|----------|
| APP_ENV | index.php | ✅ |
| APP_DEBUG | error.php | ⚠️ |
| APP_SECRET | auth.php | ✅ |

### External Services
| Variable | Files Used | Required |
|----------|------------|----------|
| REDIS_HOST | cache.php | ❌ |
| SMTP_HOST | mail.php | ❌ |

## Security Warnings
⚠️ Found hardcoded values that should be env vars:
- config.php:25 - Database password hardcoded
- api.php:10 - API key hardcoded
```

### Document Output
```markdown
# Environment Variables Documentation

## Required Variables

### DB_HOST
- **Description**: Database server hostname
- **Type**: string
- **Default**: localhost
- **Example**: `DB_HOST=localhost`

### DB_NAME
- **Description**: Database name
- **Type**: string
- **Required**: Yes
- **Example**: `DB_NAME=myapp_db`

### APP_SECRET
- **Description**: Application secret key for encryption
- **Type**: string (min 32 chars)
- **Required**: Yes
- **Security**: 🔒 Never commit to git
- **Generate**: `openssl rand -hex 32`

## Optional Variables

### APP_DEBUG
- **Description**: Enable debug mode
- **Type**: boolean
- **Default**: false
- **Warning**: ⚠️ Never enable in production
```

### Validate Output
```markdown
# Environment Validation

## Status: ⚠️ Issues Found

### Missing in .env
| Variable | Required | Default |
|----------|----------|---------|
| APP_SECRET | ✅ | - |
| REDIS_HOST | ❌ | localhost |

### Extra in .env (not in .env.example)
| Variable | Action |
|----------|--------|
| OLD_VAR | Consider removing |

### Invalid Values
| Variable | Current | Expected |
|----------|---------|----------|
| APP_DEBUG | "yes" | true/false |
| DB_PORT | "abc" | number |

## Recommendations
1. Add APP_SECRET to .env
2. Remove deprecated OLD_VAR
3. Fix APP_DEBUG value format
```

## Security Best Practices

### ห้ามทำ
```php
// ❌ Hardcode credentials
$password = "secret123";

// ❌ Commit .env to git
```

### ควรทำ
```php
// ✅ Use environment variables
$password = getenv('DB_PASS');

// ✅ Use .env.example for documentation
// ✅ Add .env to .gitignore
```

## ตัวอย่างการใช้งาน

```
/env scan              # ค้นหา env vars ทั้งหมด
/env document          # สร้าง documentation
/env validate          # ตรวจสอบความถูกต้อง
/env generate          # สร้าง .env.example
```

## Notes
- ไม่แสดงค่าจริงของ sensitive vars
- แนะนำ secure defaults
- ตรวจสอบ security issues
- สร้าง documentation อัตโนมัติ
